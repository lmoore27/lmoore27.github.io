About Me

Java was my intro into programming. I started playing minecraft shortly after it came out, and I wanted to figure out how people made mods for this game. I started watching youtube videos and tutorials until I learned that inn order to make mods I would need to learn how to program. I started getting into the basics of Java and eventually made myself a penguin in minecraft. This was my first taste of creating something using programming and it set me on my programming journey.

In high school I took computer science, this was my first time learning programming in a classroom. In this class we made simple websites, mostly having fun. It taught me a lot and this eventually led me to where I am now; learning HTML in Post-Secondary. Relearning how to create websites in HTML has been a little rough, I feel like I was better at doing this in high school but I can guaruntee that this will change by the end of this semester.

Javascript was also taught to me briefly in my High School Computer Science class. However this was super basic, we did not go farther than adding simple javascript to buttons. I did continue learning Javascript after this class was over, I worked with a library called P5.js which was a visual library for making interactive web applications. This was a fun way to learn Javascript, and got me into displaying complex objects like the mandelbrot set. These were awesome projects that got me into the basics of another programming language. This is a library that go back to often when I want to make a small project with Javascript

I learned Python in high school as well and was able to add it to my collection of programming languages. It is an awesome language that is super versatile, although it is known to be super slow. I did very basic things in high school like coding challenges online and rearranging strings. Python was another language that I revisited in my time outside of high school. I used it for web scraping and making small applications such as media players.

I am currently learning C# in a few classes. It was new to me this year and has been fun to learn. So far I have just learned the basics, but slowly getting into Windows Forms. I am hoping to translate this knowledge into making mods and maps for Rocket League. It would be a fun challenge and since Unity runs almost exclusively on C# I would have an excellent baseline in order to get right into programming and developing mods.
